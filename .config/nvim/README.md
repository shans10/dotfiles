<h1 align="center">AstroVim</h1>

**This is my personal configuration of [AstroVim](https://github.com/kabinspace/AstroVim) with some additional keybindings and aesthetic changes.** 
Visit their official gihub page for support and configuration.

<h2 align="center">Getting Started</h2>

### Windows
1. Copy the 'nvim' folder into %LOCALAPPDATA%.
2. Run `nvim +PackerSync` for the 1st time to install all plugins.
3. After the installation is completed quit nvim and start again normally just by running `nvim` in terminal.

### Linux
1. Copy the 'nvim' folder into ~/.config.
2. Run `nvim +PackerSync` for the 1st time to install all plugins.
3. After the installation is completed quit nvim and start again normally just by running `nvim` in terminal.
